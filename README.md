## 🚀 The Architecture
This platform is engineered using an elite frontend pipeline optimized for maximum capital efficiency and high-retention engagement:
- **Core Framework:** React 19 + TypeScript
- **Compilation Engine:** Vite 8 (Hot-Module Replacement)
- **3D Spatial Render:** React Three Fiber & `@react-three/drei`
- **Interface Physics:** Framer Motion (Spring-based interpolation & Scroll detection)
- **Styling Architecture:** Tailwind CSS V3

## ⚡ Deployment & Build
This repository is configured natively for **GitHub Pages** continuous deployment. 

1. Install dependencies:
   ```bash
   npm install
   ```
2. Spawn local development server:
   ```bash
   npm run dev
   ```
3. Compile production asset matrix:
   ```bash
   npm run build
   ```
   *(Note: The build command explicitly leverages `--base ./` to ensure robust asset mapping on GitHub Pages subdirectories.)*
